## The question

Most of the visualization work in this practice is built to be read precisely: a legend, an axis, an accuracy statement. This project asked the opposite question — what happens if the same satellite ocean data is treated purely as material for drawing, with no legend at all, and judged only on whether it's beautiful and honest to the underlying physics?

## Approach

The source data is public satellite-derived current velocity and sea-surface-temperature fields. Rather than plotting them as a conventional vector or heatmap, the pipeline seeds thousands of massless particles across the field and advects each one forward through time using the actual velocity data, recording its path as a single continuous line. Density of lines, not color scales, does the work of showing where flow is fast, slow, or converging — the image is a drawing made entirely out of physically real trajectories, with color reserved for temperature.

Every parameter that shapes the final image — particle density, line weight, trail length, palette — was chosen for how it reads visually, but every *line* is still a real, if simplified, integration of real current data. That constraint (aesthetic freedom in presentation, zero fabrication in the underlying paths) is what keeps the series feeling like data art rather than illustration.
